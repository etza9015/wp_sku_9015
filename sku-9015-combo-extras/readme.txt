Crea relaciones por SKU entre productos principales y extras, muestra un selector visual en producto y aplica descuentos escalonados por cantidad de extras.

== Características ==

* Relaciones dinámicas por SKU principal y SKUs de extras.
* Admin visual tipo acordeón con resumen tipo tabla.
* Buscador en tiempo real por nombre, SKU o ID.
* Tabla dinámica de extras dentro de cada relación.
* Selector visual en frontend con + / -, subtotal, descuento y total estimado.
* Descuento real aplicado únicamente a los extras relacionados.
* Soporte básico para productos simples, variaciones con SKU y variables con selector.

== Instalación ==

1. Sube el ZIP desde Plugins > Añadir nuevo > Subir plugin.
2. Activa el plugin.
3. Ve a WooCommerce > Combo por SKU.
4. Crea una relación: SKU principal -> SKUs de extras.
5. Configura los niveles de descuento, por ejemplo: 2:10, 4:15, 6:20.

== Importante ==

Desactiva reglas externas de descuento para esos mismos extras si ya usas otro plugin de pricing, para evitar descuentos duplicados.
